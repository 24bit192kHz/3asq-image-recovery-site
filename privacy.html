Privacy Policy

3asq Image Recovery

Last updated: July 18, 2026

Purpose

3asq Image Recovery is an independent Chrome extension that detects failed manga chapter images on supported 3asq domains and replaces them with exact, previously verified Wayback Machine copies when available.

Data processed by the extension

The extension reads the URLs and loading status of manga chapter images on supported 3asq domains. It does not read unrelated page content.

The extension checks a static archive URL map and domain registry hosted on this GitHub Pages site so newly verified captures and replacement 3asq domains can be used without waiting for a full extension release. GitHub receives the normal information associated with those HTTPS requests. The files contain public origins, image paths, and Wayback replay URLs only; they contain no user identifiers or executable code. If the remote map is unavailable or invalid, the extension uses its bundled map.

When a verified archived copy is available, the browser requests that image from web.archive.org. Internet Archive receives the normal information associated with a web request and handles it under its own privacy policy.

Local storage

The extension stores local counters for recovery attempts, successfully restored images, images without an archived copy, and archive loading failures. Current-page counters are stored for the browser session, while cumulative counters remain in local browser storage until the user resets them or removes the extension.

Collection, sharing, and sale

The developer does not operate a server that collects extension data. The extension does not transmit usage statistics to the developer and does not sell user data.

The extension contains no advertising, analytics, tracking tools, account system, or payment functionality. Data is not used or transferred for advertising, creditworthiness, lending, or purposes unrelated to restoring failed chapter images.

Permissions

The extension uses storage only for local recovery counters. Existing host access is limited to supported 3asq domains and is used only to detect and replace failed manga chapter images. Chrome cannot express a wildcard top-level domain such as 3asq.*, so a newly registered replacement domain requires a one-time user-approved optional host permission for that exact origin. activeTab and scripting are used only from that popup activation flow to register the packaged recovery script on the approved domain.

Remote code

The extension does not execute remote code. All JavaScript and CSS are included in the extension package. Newly verified archive mappings are downloaded only as static JSON data and are validated before use. Images and JSON mappings are data, not executable code.

User controls

Users can reset cumulative statistics from the extension popup. Removing the extension deletes its locally stored data from Chrome.
